# 请停止违法行为

侵犯公民个人信息罪（《刑法》第二百五十三条之一）‌。

组织或参与人肉搜索，非法获取、出售或提供公民个人信息，情节严重的，构成此罪。根据司法解释，「情节严重」包括信息类型（如行踪轨迹、通信内容等敏感信息）、数量（如一般信息 5,000 条以上）或违法所得数额（5,000 元以上）等标准。
